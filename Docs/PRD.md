# Product Requirements Document (PRD)

## Product Name
BookDB

## Vision
A social-first platform where reading is visible, trackable, and shareable.

## Problem
- Goodreads feels outdated
- Reading lacks a modern social identity
- No strong reader-centric social graph

## Target Users
- College students
- Startup & self-help readers
- Fiction lovers
- Habitual readers

## Core Features (MVP)
- User authentication
- Book database
- Reading status tracking
- Reviews & ratings
- User profiles
- Follow system
- Activity feed

## Non-MVP
- Audiobooks
- Payments
- Video content
- Author verification

## Success Metrics
- DAU / MAU
- Books read per user
- Reviews per book
- Follow connections
