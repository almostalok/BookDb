# App User Flow

## Landing Page (/)
- Product intro
- CTA: Login / Signup
- Featured books

## Signup (/signup)
- Name
- Email
- Password
- Google OAuth

Redirect → Onboarding

## Login (/login)
- Email
- Password
- Google OAuth

## Onboarding (/onboarding)
1. Select genres
2. Select authors
3. Follow users

## Feed (/feed)
- Books completed by followed users
- Reviews
- Milestones

## Books (/books)
- Search
- Filter by genre
- Sort by popularity / rating

## Book Detail (/book/[id])
- Cover
- Title
- Author
- Rating
- Description
- Reading status
- Reviews

## Profile (/user/[username])
- User info
- Books read count
- Reading lists
- Reviews
- Followers

## Collections (/collections)
- Create collections
- Add/remove books
- Public / Private

## Messages (/messages)
- 1–1 chat
- Text only (MVP)
